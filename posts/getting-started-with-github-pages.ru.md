GitHub Pages — отличный способ бесплатно размещать статические веб-сайты. В этом посте я проведу вас через процесс развертывания простого блога, подобного этому.

## Шаг 1: Создание репозитория

Сначала создайте новый репозиторий на GitHub. Вы можете назвать его как угодно, но если вы хотите, чтобы он был доступен по адресу `username.github.io`, назовите его точно `username.github.io`.

## Шаг 2: Загрузка файлов

Клонируйте репозиторий локально, добавьте ваши HTML, CSS и JavaScript файлы, затем загрузите их на GitHub:

```bash
git clone https://github.com/username/username.github.io.git
cd username.github.io
# Добавьте ваши файлы
git add .
git commit -m "Initial commit"
git push origin main
```

## Шаг 3: Включение GitHub Pages

Перейдите в настройки вашего репозитория, прокрутите вниз до раздела "Pages" и выберите ветку, которую хотите развернуть (обычно `main`). Ваш сайт будет доступен через несколько минут!

Вот и все! Ваш блог теперь доступен в интернете. Вы можете получить к нему доступ по адресу `https://username.github.io`.
