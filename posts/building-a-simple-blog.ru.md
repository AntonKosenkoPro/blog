Создание блога не должно быть сложным. Иногда самые простые решения — лучшие. В этом посте я объясню подход, который использовал для создания этого блога.

## Почему простота?

Существует множество фреймворков для блогов и вариантов CMS, но иногда вам просто нужно что-то легкое и быстрое. Этот блог использует:

- Простой HTML для структуры
- CSS для стилизации
- Ванильный JavaScript для интерактивности

## Ключевые особенности

Блог включает:

- Адаптивный дизайн, работающий на всех устройствах
- Чистый, современный стиль
- Легкое добавление постов (просто добавьте в массив JavaScript)
- Быстрая загрузка
- **Поддержка Markdown** для написания постов
- **Многоязычность** — переключение между русским и английским

## Добавление новых постов

Чтобы добавить новый пост в блог, просто добавьте объект в массив `blogPosts` в `script.js`. Каждому посту нужно:

- `id`: Уникальный идентификатор (используется в URL)
- `title`: Заголовок поста (объект с `en` и `ru`)
- `date`: Дата публикации
- `excerpt`: Краткое описание (объект с `en` и `ru`)
- Создайте файлы `post-id.en.md` и `post-id.ru.md` в папке `posts/`

## Написание на Markdown

Вы можете писать свои посты на Markdown! Это значительно упрощает форматирование контента. Вот несколько примеров:

- **Жирный текст** с помощью `**жирный**`
- *Курсивный текст* с помощью `*курсив*`
- Блоки кода с тройными обратными кавычками
- Списки, заголовки, ссылки и многое другое

Это так просто! Никакой базы данных, никакой сложной настройки, только чистые веб-технологии.
